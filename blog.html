Question-1: What are the different ways to select an element in the DOM?

There are many ways to select elements in the DOM. Such As: 1. getElementsByTagName() 2. getElementsByClassName() 3. getElementById() 4. querySelector() 5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

By innerHTML we can get the HTML content inside an element. By innerText we get the visible text inside an element without tags. By textContent we get the entire text content inside an element without tag.

Question-3: What is event delegation in the DOM?

event delegation" is a technique where we can attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element.

Question-4: What is event bubbling in the DOM?

Event Bubbling is when an event starts from the clicked element and moves up through its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

1. For create elements - createElement(), 2. For add elements - appendChild(), prepend() 3. For remove elements - remove(), removeChild()
